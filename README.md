# Socio Constructor

Página web profesional para Socio Constructor - Seguridad y Confianza

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
npm start
```
